# WebArcade

A lightweight plugin platform for building native desktop applications with **SolidJS** and **Rust**.

- ~2.5 MB binary (uses system WebView)
- Dynamic plugin loading at runtime
- Optional Rust backend per plugin (DLL)

## Installation

### Prerequisites

1. **Rust** - https://rustup.rs/
2. **Bun** - https://bun.sh/

### Quick Start

```bash
# Install the CLI
cargo install webarcade

# Create a new project
webarcade init my-app
cd my-app

# Run in development mode
webarcade dev

# Build production app
webarcade app
```
